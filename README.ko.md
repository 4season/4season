## 안녕하세요, 허준호입니다 👋

저는 계산신경과학, 전뇌 네트워크 모델링, 인간의 학습과 의사결정, 계산정신의학에 관심을 가진 컴퓨터공학 학부연구생입니다.

주의와 각성, 학습 효율이 변화하는 경험을 통해 같은 사람도 시간에 따라 정보를 다르게 처리하는 이유와, 비슷한 조건을 가진 사람들 사이에서도 서로 다른 인지적·행동적 양상이 나타나는 이유에 관심을 갖게 되었습니다. 개인적 경험을 과학적 근거로 일반화하지 않고, 검증할 연구 질문의 출발점으로 삼아 계산모델과 행동·신경데이터를 통해 탐구하고자 합니다.

## 연구 관심 분야

- 계산신경과학
- 전뇌 네트워크 모델링
- 뇌영상 및 신경데이터 분석
- 인간의 학습·일반화·의사결정
- 계산정신의학

## 현재 연구 경험

현재 백석대학교 Brain–AI Interface Lab에서 학부연구생으로 활동하고 있습니다.

- 계산신경과학 및 신경데이터 분석 관련 논문 검토
- 저널 토론 참여
- 신경데이터 전처리의 기초 워크플로 학습
- 초기 연구 질문, 프로젝트 방향 및 분석 환경 마련에 참여

## 현재 연구 방향

특히 다음 질문에 관심이 있습니다.

- 주의와 각성이 학습·추론·의사결정에 어떤 영향을 미치는가
- 뇌가 감각 증거와 사전지식을 어떻게 통합하는가
- 뇌 네트워크의 상호작용에서 학습과 일반화가 어떻게 나타나는가
- 신경 계산의 변화가 행동의 차이와 정신병리학적 증상으로 어떻게 이어지는가
- 계산모델로 개인 내 변동성과 개인차를 어떻게 함께 설명할 수 있는가

## 주요 프로젝트

### [TrendXiv](https://github.com/4season/TrendXiv)

약 82,000편의 arXiv 컴퓨터과학 논문과 인용정보를 활용한 연구 동향 예측 파이프라인입니다. 시간 분할 검증, 데이터 누출 방지 및 모델 일반화 평가에 중점을 두었습니다.

### [ConvDDI](https://github.com/4season/ConvDDI)

알약 이미지 분류 결과를 성분 및 병용금기 데이터베이스와 통합한 CNN 기반 의약품 분류·약물상호작용 탐지 파이프라인입니다.

### [sobdm-project](https://github.com/4season/sobdm-project)

운행 일정, 교통, 날씨와 주기적 시간 특성을 활용한 MLP 기반 버스 도착 지연 예측 프로젝트입니다.

### [Magnolia](https://github.com/4season/Magnolia)

Go, Rust, Kotlin으로 개발한 백엔드 및 알림 처리 시스템입니다.

### [junho.art](https://github.com/4season/junho.art)

HTML, CSS, JavaScript와 GitHub API 연동으로 개발한 개인 연구 포트폴리오 웹사이트입니다.
