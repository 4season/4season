## 안녕하세요, 허준호입니다 👋
저는 계산신경과학(Computational Neuroscience), 계산정신의학(Computational Psychiatry), 그리고 인간의 학습과 의사결정에 관심을 가진 컴퓨터공학 학부생입니다.
저의 학문적 관심은 다음과 같은 질문에서 시작되었습니다.
왜 사람들은 같은 정보를 받아도 서로 다르게 이해하고, 다른 속도로 학습하며, 서로 다른 결정을 내릴까?
저는 이 질문에 대해 계산 모델, 행동 데이터, 그리고 신경 데이터를 결합하여 접근하고자 합니다.

---

### 연구 관심 분야

* 계산신경과학
* 계산정신의학
* 강화학습과 의사결정
* 인간 학습 및 인지 모델링
* 행동 및 신경 데이터 분석

### 현재 관심 분야

저는 머신러닝 모델 구현, 데이터 전처리, 백엔드 개발, 클라우드 인프라 구축 프로젝트를 수행하며 소프트웨어와 데이터 시스템에 대한 경험을 쌓아왔습니다.
현재는 다음 분야를 중심으로 학습하고 있습니다.

* 강화학습
* 확률 모델링
* 계산신경과학
* 행동 및 신경 데이터 분석
* 뇌와 인지 과정의 수학적 모델링

장기적으로는 주의(attention), 각성(arousal), 학습(learning), 의사결정(decision-making)이 어떻게 신경 계산(neural computation)으로부터 나타나는지, 그리고 이러한 과정이 정신질환에서 어떻게 변화하는지를 이해하는 연구자가 되고자 합니다.
